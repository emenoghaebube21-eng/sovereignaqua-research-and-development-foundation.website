Overview

Welcome to the SovereignAqua Research & Development Foundation platform.

This repository contains the source code, documentation, and supporting resources for the Foundation's public website and digital management platform.

The project is being developed using modern web technologies while following software engineering best practices, including modular architecture, reusable components, semantic HTML, responsive design, and structured documentation.

Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* Decap CMS
* GitHub
* GitHub Pages (or other hosting provider)

Repository Structure

```text
admin/
assets/
content/
data/
docs/
uploads/
```

Development Goals

* Responsive institutional website
* Research portal
* Project showcase
* Asset Enrolment platform
* Administration dashboard
* Content Management System
* Secure and scalable architecture

Coding Standards

* Use semantic HTML.
* Write modular CSS.
* Organize JavaScript into reusable modules.
* Maintain consistent file naming.
* Document all major architectural decisions.

Documentation

Project documentation is stored in the `docs/` directory.
